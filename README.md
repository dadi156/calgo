# cTrader Free Indicators

A collection of free and open-source indicators for the cTrader platform.

This repository contains various technical analysis tools covering multiple domains, including market structure, volume analysis, moving averages, regression models, volatility, and custom utilities.

All indicators are provided as-is and intended for learning, experimentation, and practical use.

---

## Available Indicators

### Structure / Levels
- Fair Value Gap (Extended)  
- Pivot Points  
- Round Numbers  

### Volume
- Volume Activity Profiler  
- Volume Profile  
- Volume Spread Analysis  
- VWAP (Volume Weighted Average Price)  

### Moving Averages
- Anchored Moving Average  
- Deviation-Scaled Moving Average  
- Instantaneous Trendline  
- Kaufman Adaptive Moving Average  
- Moving Average Channel  
- Moving Averages Suite  
- Trend Channel Moving Average  

### Regression-Based
- Advanced Regression Channel  
- Linear Regression Channel  
- Nadaraya-Watson Kernel Regression  

### Volatility
- Initial Balance  
- Trend Volatility Trail  

### Oscillators / Filters
- Balance of Power  
- [Price-Time Filtering](https://ctrader.com/products/3134)
- Waddah Attar Explosion  

---

## Notes

- These indicators focus on data transformation and visualization, not signal generation.
- No performance claims or trading guarantees are implied.
- Most tools are designed to be combined with your own market structure or price action framework.

---

## Usage

Each folder contains a standalone indicator for the cTrader platform.  
You can compile them using cTrader Automate (cAlgo).
